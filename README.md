# Entry-note : A CRUD App with Vanilla JS & Capacitor JS

This is a simple CRUD (Create, Read, Update, Delete) application built using Capacitor JS and Vanilla JS. The app allows you to manage note entries and track the total of credit and debit entries.

## Features

- **Add Note Entry:** You can create a new note entry.
- **Delete Entry:** Existing entries can be removed.
- **Edit Entry:** Modify the content of any note entry.
- **Entry update history:** View the update history of any entries.
- **Total Credit and Debit Tracking:** The app calculates and displays the total amount of credit and debit entries.

## Getting Started

To get a local copy up and running, follow these simple steps:

1. **Clone this repo:** Download the repo by running git clone https://github.com/semilore8/capacitor-entry-note-app.git
2. **Install all dependencies** Run `npm install` to install all dependencies.
3. **Add Android or IOS:** Use `npx cap add android` or `npx cap add ios` to add Android or IOS to your project.
4. **Run the app on web:** Run `npm start` to run a live preview on the web.
5. **Run the preview on android:** Run `npm run android` to run the app on a android emulator or a device.

## Usage

Once your app is set up, you can start adding, deleting, and editing note entries. The app will automatically calculate and display the total of your credit and debit entries.
